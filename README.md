## Description

A simple BackEnd system to start queues, with one user as admin, and another as a client.

## Installation

```
Download the repo, after downloading it, access it with:
```

```bash
$ cd SMS_NestJS
```

`Then run this command:`

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
